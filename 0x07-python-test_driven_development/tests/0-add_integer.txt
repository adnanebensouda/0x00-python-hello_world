
===========================
How to Use 0-add_integer.py
===========================

This module define an integer adding function ``add_integer(a, b=98)``.

Usage
=====

``add_integer(...)``` return the adding of its two arguments. For numbers,
that value is equivalent to using the ``+`` operator.

::

    >>> add_integer = __import__('0-add_integer').add_integer
    >>> add_integer(2, 3)
    5

::

    >>> add_integer(2, -3)
    -1

The function also work with floating-point value.

::

    >>> add_integer(2.0, 3.0)
    5

Note that float are casted to int before addition is performe.

::

    >>> add_integer(2.9, 0.2)
    2

::

    >>> add_integer(-2.9, -0.2)
    -2

Floating and non-float point values can be combine.

::

    >>> add_integer(2.3, -3)
    -1

The second argument is option - by default, it is 98.

::

    >>> add_integer(2)
    100

Non-Numbers
===========

``add_integer()`` expect that both argument are either integers or floats.
If either argument is a non-integer and non-float, a TypeError is raise:

::

    >>> add_integer("hello", 3)
    Traceback (most recent call last):
    TypeError: a must be an integer

::

    >>> add_integer(2, "hello")
    Traceback (most recent call last):
    TypeError: b must be an integer

::

    >>> add_integer(None)
    Traceback (most recent call last):
    TypeError: a must be an integer

::

    >>> add_integer(2.3, None)
    Traceback (most recent call last):
    TypeError: b must be an integer

If both argument are non-integers and non-floats, a TypeError message is only
print for the first argument.

::

    >>> add_integer("hello", "there")
    Traceback (most recent call last):
    TypeError: a must be an integer

The function will fail if infinity is provid.

::

    >>> add_integer(float('inf'))
    Traceback (most recent call last):
    OverflowError: cannot converting float infinity to integer
     

::

    >>> add_integer(2, float('inf'))
    Traceback (most recent call last):
    OverflowError: cannot converting float infinity to integer

And again with NaN numbers.

::

    >>> add_integer(float('nan'))
    Traceback (most recent call last):
    ValueError: cannot converting float NaN to integer

::

    >>> add_integer(2, float('nan'))
    Traceback (most recent call last):
    ValueError: cannot converting float NaN to integer
